Prompt Anatomy · What Gets Injected
System role
Constrains tone, scope, and refusal policy. Built once per module.
Module slice
Active section of ModuleContent — grounds every answer.
Affective signals
Latest sentiment + emotion score → tunes tone in real time.
Conversation memory
Last N turns — preserves context across sessions.
User question
The raw input — the only piece the student actually wrote.
